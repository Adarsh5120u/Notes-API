# Notes API

A simple RESTful Notes API built with **Node.js** and **Express**.  
This API allows users to create, read, update, and delete notes using HTTP endpoints.

---

## Features

- Create a new note
- Get all notes
- Get a note by ID
- Update a note
- Delete a note

This project demonstrates basic **REST API design**, **Express routing**, and **controller structure**.

---

## Tech Stack

- Node.js
- Express.js
- JavaScript
- Thunder Client / Postman (for API testing)
